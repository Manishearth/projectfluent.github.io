Fluent is a localization paradigm designed to unleash the expressive power of the natural language.

Visit GitHub →

For localizers

  • Natural-sounding translations
  • Number and date formatting
  • Plural rules
  • Genders
  • Grammatical cases
  • Readable file format

For developers

  • Client- and server-side
  • Simple API
  • Good error handling
  • Integration with ICU, ECMA 402
  • Bi-directional text
  • Open-source (Apache 2.0)
  • Implemented in JavaScript, Python and Rust
  • Well-defined EBNF grammar
